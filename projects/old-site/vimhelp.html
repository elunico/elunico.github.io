Pressing <escape> will open command mode. You can then type certain commands and use <enter> to execute them. A list of possible commands is found below.

Commands

    Color Scheme Changes

  • tdm - Toggle Dark Mode (this will reset all custom colors to defaults!)
  • rcs - Reset color scheme (resets all colors and uses whatever mode is specified by (prefers-color-scheme: dark))
  • gmc - Get main color scheme color. Shown in command status area.
  • gac - Get accent color scheme color. Shown in command status area.
  • gtc - Get text color scheme color. Show in command status area
  • smc - Set main color scheme color. Color should be '#RGB' or '#RRGGBB' color.
  • sac - Set accent color scheme color. Color should be '#RGB' or '#RRGGBB' color.
  • stc - Set text color scheme color. Color should be '#RGB' or '#RRGGBB' color.

  • gsto - Show the number of seconds the status area stays visible on command input
  • ssto - Set the number of seconds the status area stays visible on command input

  • rm zsh_history - clear inter-page command history
  • cls - calls localStorage.clear(). This will affect other websites!!

    • Page Navigation

  • ->c - Go to color picker (wow how convenience; see above)
  • ->h - Go to page Home
  • ->p - Go to page Programming
  • ->l - Go to page Linguistics
  • ->i - Go to page Info
  • ? - Navigate to this help page

    • Scrolling Commands

  • ]] - Scroll to next section
  • [[ - Scroll to previous section
  • gg - Scroll to page beginning
  • G - Scroll to page end

    • Navigation commands

  • :q - Quit website
  • :q! - Forcibly quit website

    • History

  • up arrow - fill in the previously issued command
  • down arrow - fill in the subsequently issued command
Note: when you have the vim cursor blinking you can paste. You can also copy by selecting the text as normal.

Font Alterations

This website allows you to customize the font that you view it in. Once you activate command mode by pressing <escape> as stated above, you can enter the following commands to change the fonts of the webpage.

    Browser & Site Defaults

  • f reset - Set font to website default
  • f dflt-s - Set font to browser's default serif font
  • f dflt-ss - Set font to browser's default sans-serif font
  • f dflt-m - Set font to browser's default monospace font

    • Propriety Fonts

  • f c - Set font to Consolas†
  • f mon - Set font to Monaco†

    • Web fonts

  • f mdj - Set font to Menlo/DejaVu Sans*
  • f mdj - Set font to Mononoki
  • f um - Set font to Ubuntu Mono
  • f scp - Set font to Source Code Pro
  • f sm - Set Font to Space Mono
  • f i - Set font to Inconsolata
  • f crpr - Set font to Courier Prime
  • f fs - Set Font to Fontdiner Swanky
  • f ua - Set Font to Uncial Antiqua
  • f samt - Set Font to Sacramento
  • f se - Set Font to Special Elite

    • Custom Font Family

  • cf <family-name> - Set the font-family property to whatever is given after the cf.
† These fonts are propriety. Font file must be present on the local machine for this to work.
* Menlo is propriety. If Menlo is not present, DejaVu Sans Mono is used.
